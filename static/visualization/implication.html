Implication Similarity

Formula used to calculate similarity using this method:

$$ SIM(A, B) = f(A \equiv_= B) $$

where f represents aggregator function and \(A \equiv_= B\) represents the formula:

$$ A \equiv_= B = [\mu_A(a_1) \equiv_= \mu_B(b_1), ..., \mu_A(a_n) \equiv_= \mu_B(b_n)] $$

and

$$ a \equiv_= b = t(a \Rightarrow b, b \Rightarrow a) $$

where t is a t-norm and \(\Rightarrow\) represents fuzzy implication. Fuzzy Implication \(\Rightarrow\) is defined as

$$ a \Rightarrow b = s(v(a), b)$$

s represent a t-konorm, and \(v\) a fuzzy negation. All implications implemented here use the following negation: \(v(a) = 1 - a\). Łukasiewicz implication use Łukasiewicz t-konorm. Implication named maximum use max t-konorm.
