Angular distance

Formula used to calculate similarity using this method:

$$ SIM_{cos\alpha}(A, B) = \frac{\sum\limits_{u \in U} \mu_A(u) \mu_B(u)}{\sqrt{\sum\limits_{u \in U} \mu_A(u)^2} \sqrt{ \sum\limits_{u \in U} \mu_B(u)^2}} $$

Note that if the denominator of given fraction is equals to zero this method implementation will return 0 as result.
