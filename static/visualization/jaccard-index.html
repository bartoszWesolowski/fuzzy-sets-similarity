Jaccard Similarity

Formula used to calculate similarity using this method:

$$ SIM(A, B) = \frac{g(A \cap B) + \gamma g(\bar{A} \cap \bar{B})}{g(A \cap B) + \alpha g(A \cap \bar{B}) - \beta g(\bar{A} \cap B)} $$

For \(\alpha, \beta, \gamma \geq 0\). Function g represents scalar evaluator of fuzzy set.
